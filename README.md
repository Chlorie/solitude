# Solitude

Implementation of human-like sudoku solving strategies.

## List of Sudoku solving techniques

- [x] single
- [x] intersection
- [x] pair
- [ ] x-wing
- [x] triple
- [ ] single digit 2-length chain, finned x-wing
- [ ] xy-wing, w-wing
- [ ] swordfish, jellyfish, finned or not
- [ ] x-chain, xy-chain
- [ ] arbitrary chains
- [ ] ALS
- [ ] more insane fish
- [ ] fish with chains
- [ ] forcing chains, forcing nets
- [x] brute force
